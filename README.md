## Projects

### Net Analyzer Pro
Android network analysis and OSINT toolkit with 10+ tools including IP lookup, port scanner, ping, MAC vendor lookup, username OSINT (55+ platforms), Google Dork generator, DNS leak audit, and admin panel finder.

`Flutter` `Dart` `Android` `OSINT` `Network Analysis`

---

### Payroll Pro ID
Indonesian payroll calculator implementing PPh 21 TER per PMK 168/2023, BPJS Kesehatan & Ketenagakerjaan, batch processing, Excel/PDF slip export, and SHA-256 checksums. Built with Streamlit.

`Python` `Streamlit` `PPh 21` `BPJS` `openpyxl` `reportlab`

---

### Sistem Presensi 2FA
Mobile attendance system with 2FA, GPS geofencing, selfie + liveness detection, QR scan, and anti-spoofing (root/emulator/mock GPS detection). Laravel 11 backend with Next.js admin dashboard.

`Flutter` `Laravel` `PostgreSQL` `Redis` `Next.js` `GPS Geofencing`

---

### Pengadaan Publik Tracker
Civic tech tool for detecting anomalies in Indonesian government procurement (pengadaan barang/jasa) data. Analyzes 11,461 RUP packages across 38 SKPD in Tangerang Selatan — detects Penunjukan Langsung clustering, near-threshold stacking, and budget concentration patterns.

`Python` `Pandas` `Playwright` `Civic Tech` `Open Data`

---

### Mitra GKMZ Kaltim
Supply chain management app for palm oil cooperative (Gapoktan) in East Kalimantan. Handles SPB document generation with QR codes, role-based access for 923+ farmer records across three cooperatives.

`Flutter` `Provider` `SQLite` `QR Code` `Supply Chain`

---

### Surat Peringatan Generator
Python mail merge automation for generating Surat Peringatan (SP warning letters) in bulk from a Word template. Supports batch CSV input and Windows `.bat` launcher. Deployed internally at Hygiene Kleen.

`Python` `Word Mail Merge` `lxml` `HR Automation`

---

### HR Dashboard
Local web app for HR management of 1,000+ employees. Features employee CRUD, payroll calculation (PPh 21 TER, BPJS), document generation, role-based access control, Excel/CSV import, and audit logging.

`FastAPI` `React` `PostgreSQL` `RBAC` `HR Tech`
